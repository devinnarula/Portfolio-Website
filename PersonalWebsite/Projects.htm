Projects

Clemson Canvas Alexa Skill

I created this skill for an Amazon Alexa with a team of three others. It can
access Canvas, the service Clemson uses for assignments, to read the user
their upcoming assignments. It won best hardware hack in the 2019 HelloWorld
Hackathon event at Clemson University.


Puddle iOS Application

This application was created in a team with three others. It functions as a
pedestrian map application to alert people of obstacles in the area such as
puddles. My role was to create the interface of the app while the other group
members worked on the backend configurations. It won best use of Radar.io in
the 2019 CUhackit hackathon event at Clemson University.
